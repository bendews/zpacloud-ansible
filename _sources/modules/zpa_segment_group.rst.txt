.. _zpa_segment_group_module:


zpa_segment_group -- Create a Segment Group
===========================================

.. contents::
   :local:
   :depth: 1


Synopsis
--------

This module will create/update/delete a segment group resource.







Parameters
----------

  client_id (type: str)
    


  client_secret (type: str)
    


  customer_id (type: str)
    


  applications (type: list)
    


    id (required, type: str)
      


    name (type: str)
      



  config_space (type: str, default: DEFAULT)
    


  description (type: str)
    


  enabled (type: bool)
    


  id (type: str)
    


  name (required, type: str)
    


  policy_migrated (type: bool)
    


  tcp_keep_alive_enabled (type: str)
    


  state (type: str, default: present)
    









Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create/Update/Delete a Segment Group
      willguibr.zpacloud.zpa_segment_group:
        name: Example Segment Group
        config_space: "DEFAULT"
        description: Example Segment Group
        enabled: true
        policy_migrated: true
        tcp_keep_alive_enabled: "1"
