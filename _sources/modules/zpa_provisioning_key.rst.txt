.. _zpa_provisioning_key_module:


zpa_provisioning_key -- Create a Provisioning Key.
==================================================

.. contents::
   :local:
   :depth: 1


Synopsis
--------

This module will create/update/delete a specific Provisioning Key by association type (CONNECTOR_GRP or SERVICE_EDGE_GRP).







Parameters
----------

  client_id (type: str)
    


  client_secret (type: str)
    


  customer_id (type: str)
    


  app_connector_group_id (type: str)
    


  app_connector_group_name (type: str)
    


  creation_time (type: str)
    


  enabled (type: bool, default: True)
    


  expiration_in_epoch_sec (type: str)
    


  id (type: str)
    


  ip_acl (type: str)
    


  max_usage (required, type: str)
    


  modified_by (type: str)
    


  modified_time (type: str)
    


  name (required, type: str)
    


  provisioning_key (type: str)
    


  enrollment_cert_id (required, type: str)
    


  enrollment_cert_name (type: str)
    


  ui_config (type: str)
    


  usage_count (type: str)
    


  zcomponent_id (required, type: str)
    


  zcomponent_name (type: str)
    


  association_type (required, type: str)
    


  state (type: str, default: present)
    









Examples
--------

.. code-block:: yaml+jinja

    
    - name: Get ID Information of a Service Edge Group Enrollment Certificate
      willguibr.zpacloud.zpa_enrollement_certificate_info:
        name: "Service Edge"
      register: enrollment_cert_service_edge

    - name: Get ID Information of an Service Edge Group
      willguibr.zpacloud.zpa_service_edge_groups_info:
        name: "Example"
      register: service_edge_group
    - name: "Create/Update/Delete App Connector Group Provisioning Key"
      willguibr.zpacloud.zpa_provisioning_key:
        name: "App Connector Group Provisioning Key"
        association_type: "CONNECTOR_GRP"
        max_usage: "10"
        enrollment_cert_id: "{{ enrollment_cert_service_edge.data[0].id }}"
        zcomponent_id: "{{ service_edge_group.data[0].id }}"
