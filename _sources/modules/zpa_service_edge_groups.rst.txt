zpa_service_edge_groups -- Create a Service Edge Group.
=======================================================

.. contents::
   :local:
   :depth: 1


Synopsis
--------

This module create/update/delete a Service Edge Group resource in the ZPA Cloud.







Parameters
----------

  client_id (type: str)
    


  client_secret (type: str)
    


  customer_id (type: str)
    


  city_country (type: str)
    


  country_code (type: str)
    


  description (type: str)
    


  enabled (type: bool, default: True)
    


  geolocation_id (type: str)
    


  id (type: str)
    


  is_public (type: str)
    


  latitude (type: str)
    


  location (type: str)
    


  longitude (type: str)
    


  name (required, type: str)
    


  override_version_profile (type: bool)
    


  upgrade_day (type: str, default: SUNDAY)
    


  upgrade_time_in_secs (type: str, default: 66600)
    


  version_profile_id (type: str)
    


  version_profile_name (type: str)
    


  version_profile_visibility_scope (type: str, default: NONE)
    


  service_edges (type: list)
    


    id (type: str)
      


    name (type: str)
      



  trusted_networks (type: list)
    


    id (type: str)
      


    name (type: str)
      



  state (type: str, default: present)
    









Examples
--------

.. code-block:: yaml+jinja

    
    - name: Create/Update/Delete a Service Edge Group
      willguibr.zpacloud.zpa_service_edge_groups:
        name: "Example"
        description: "Example2"
        enabled: true
        city_country: "California, US"

        latitude: "37.3382082"
        longitude: "-121.8863286"
        location: "San Jose, CA, USA"
        upgrade_day: "SUNDAY"
        upgrade_time_in_secs: "66600"
        override_version_profile: true
        version_profile_id: "0"
        dns_query_type: "IPV4"
